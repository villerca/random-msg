# npm-random-msg

# mensajes aleatorios

Cada vez que empiezo un nuevo proyecto, no sé qué escribir para la primera confirmación. Después de hacer un "git init" técnicamente no hay nada allí...

## Instalar

```npm
npm install -g random-msg
```

# Uso

```bash
mensaje aleatorio
```
